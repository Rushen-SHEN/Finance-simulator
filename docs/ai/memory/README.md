# Session Memory

## 必须包含的字段

```markdown
# YYYY-MM-DD Session Log

## Goal
[本次会话目标]

## What Changed
- [变更 1]
- [变更 2]

## What Was Validated
- [验证命令 + 结果]

## Files Touched
- [文件路径列表]

## Risks / Blockers
- [如有]

## Commit SHA
`xxxxxxx`

## Next Entry Point
[下次从哪里继续]
```
